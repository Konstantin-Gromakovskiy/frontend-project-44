## Installing

Using npm:
```bash
$ git clone git@github.com:Konstantin-Gromakovskiy/frontend-project-44.git
```
```bash
$ npm install 
```

Brain even
--------------------------  
You can start the game by entering the command:
 ```bash
$ brain-even
```
